# 🎨 SoftPay 微交互实现说明

## 概述

为 SoftPay 项目实现了三个高级触觉微交互，赋予界面真实的纸质/文具质感，完美匹配极简 Japandi/Notion 美学风格。

---

## ✅ 已实现的三大微交互

### 1. 📮 触觉印章按压物理效果（Tactile Stamp Press Physics）

**目标元素：**
- 三个打赏等级卡片（$1 绿茶杯、$3 戚风蛋糕、$5 黑胶唱片机）
- 自定义金额卡片
- 主提交按钮（"Simulate cozy tea break"）

**实现效果：**
- **悬停状态**：元素轻微上浮（y: -2px）并放大（scale: 1.01）
- **点击状态**：模拟真实印章按压
  - 缩小到 96%（scale: 0.96）
  - 轻微旋转（rotate: -0.5deg）
  - 向下移动 2px（y: 2）
  
**物理参数：**
```typescript
transition={{
  type: "spring",
  stiffness: 500,  // 高刚度，消除漂浮感
  damping: 15,     // 适度阻尼，快速回弹
  mass: 0.5        // 轻质量，响应迅速
}}
```

**用户体验：**
- 点击时有明确的"顿挫感"
- 模拟真实印章盖章的触感
- 快速响应，无延迟感
- 完美匹配 Notion 风格的干净利落

---

### 2. 📬 重力投递与明信片摇摆入场（Gravity Drop & Postcard Wobble Ingress）

**目标元素：**
- 明信片模态框（Generative Receipt Postcard）

**实现效果：**
- **入场动画**：从屏幕顶部掉落
  - 初始位置：屏幕上方（y: -100vh）
  - 初始状态：透明（opacity: 0）、倾斜（rotate: -6deg）、缩小（scale: 0.9）
  
- **落地动画**：模拟重物落入邮箱
  - 最终位置：屏幕中央（y: 0）
  - 最终状态：完全显示（opacity: 1）、轻微倾斜（rotate: -1.5deg）、正常大小（scale: 1）
  
- **弹性效果**：落地后有轻微弹跳
  - 模拟厚重纸卡撞击边界后的动能残留
  - 自然的摇摆稳定过程

**物理参数：**
```typescript
transition={{
  type: "spring",
  stiffness: 220,  // 中等刚度，允许弹跳
  damping: 14,     // 低阻尼，产生弹性
  mass: 0.8        // 较重质量，模拟纸卡重量
}}
```

**用户体验：**
- 明信片像真实物体一样从上方掉落
- 落地时有真实的物理反馈
- 轻微的摇摆增加趣味性
- 强化"收到明信片"的仪式感

### 3. ✨ SSR 镀金流光反射（SSR Gilded Shimmer Reflection）

**目标元素：**
- 明信片模态框（仅当稀有度为 SSR 时）
- 邮箱收藏卡片（仅当稀有度为 SSR 时）

**实现效果：**
- **高端金箔反射**：模拟阳光下的金箔质感
- **流动光泽**：光线缓慢流过边框
- **哑光金属感**：避免廉价的霓虹发光效果

**CSS 动画：**
```css
@keyframes gildedFlow {
  0% { background-position: 0% 50%; }
  50% { background-position: 100% 50%; }
  100% { background-position: 0% 50%; }
}
```

**渐变配置：**
```css
background: linear-gradient(
  90deg,
  transparent,
  rgba(254, 240, 138, 0.3) 50%,  /* 柔和的金色 */
  transparent
);
background-size: 200% 200%;
animation: gildedFlow 4s linear infinite;
```

**用户体验：**
- SSR 卡片立即识别（视觉层级明确）
- 高级感和稀有感
- 不刺眼，符合极简美学
- 4 秒循环，优雅缓慢

## 🎯 设计原则

### 1. 物理真实性
- 所有动画都基于真实物理规律
- 使用 Framer Motion 的 spring 物理引擎
- 参数精心调校，消除"漂浮感"

### 2. 触觉反馈
- 每个交互都有明确的视觉反馈
- 模拟真实世界的触感
- 增强用户的操作确定性

### 3. 极简美学
- 动画克制，不过度
- 符合 Notion/Japandi 风格
- 黑白配色为主，金色点缀

### 4. 性能优化
- 使用 CSS transform（GPU 加速）
- 避免触发 layout reflow
- 动画流畅，60fps

## 📊 技术实现

### 使用的技术栈

**Framer Motion：**
- `motion.button` - 按钮动画
- `motion.div` - 容器动画
- `AnimatePresence` - 进出场动画
- `whileHover` / `whileTap` - 交互状态
- `initial` / `animate` / `exit` - 动画状态
- `transition` - 物理参数配置

**CSS Animations：**
- `@keyframes` - 关键帧动画
- `animation` - 动画应用
- `::before` 伪元素 - 叠加层
- `mask-composite` - 遮罩合成

**Tailwind CSS v4：**
- 动态类名组合
- 条件样式应用
- 响应式设计
